# 个人博客

基于[VuePress](https://vuepress.vuejs.org/)的个人博客

## 开始

```bash
# 安装依赖
# 不建议使用 npm 安装，官方说明如下
# 注意：如果你的现有项目依赖了 webpack 3.x，推荐使用 Yarn 而不是 npm 来安装 VuePress。因为在这种情形下，npm 会生成错误的依赖树。
yarn install

# 运行
yarn docs:dev

# 生产
# 默认情况下，文件将会被生成在 .vuepress/dist
yarn docs:build
```
